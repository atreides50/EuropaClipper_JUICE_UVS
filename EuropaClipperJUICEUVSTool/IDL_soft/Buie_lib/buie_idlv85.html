Changes to Buie IDL v8.5 Library

This is a reverse chronological list of changes to all library routines since the switch-over from the previous version of IDL.

edgefit: 2017/01/19, MWB, many rewrites, may sort of work now but still a bit fragile or non-general. Use with extreme caution.

upsample: 2017/01/13, MWB, seems to be working now (still lots of debugging output)

spexpand: 2017/01/12, MWB, added ROOT2 keyword option

mcmcsamp: 2017/01/06, modified to allow imposing restricted domains for sampling

reconstmail: 2017/01/03, MWB, refinements to report, less detail if event is not close

idstring: 2016/12/31 - Written by Marc W. Buie, Southwest Research Institute

subarr: 2016/12/28, MWB, added byte input option for FILL

gauss1d: 2016/12/27, Written by Marc W. Buie, Southwest Research Institute

mcmcsamp: 2016/12/22, Written by Marc W. Buie, Southwest Research Institute with input from Alex Parker.

wedge: 2016/12/15, MWB, added NCOLORS keyword, this changes the default behavior very slightly.

reconstmail: 2016/12/12, MWB, added campaign result summary

reconnotice: 2016/12/07, Written by Marc W. Buie, Southwest Research Institute

mailmsg: 2016/12/05, MWB, added REPLYTO keyword

reconstmail: 2016/11/30, MWB, added reminder email

mailmsg: 2016/11/30, MWB, added BCCADDR keyword

showsrc: 2016/11/22, MWB, allow byte input images

reconstmail: 2016/11/17, Written by Marc W. Buie, Southwest Research Institute

mysqlquery: 2016/11/14, MWB, fix to LL format code handling, propagated from readcol BUGS/WISH LIST: Ought to verify connection to MySQL server. Does nothing helpful with SQL command results. Does nothing helpful to identify/report bad SQL syntax.

rdstarc: 2016/11/05, MWB, added support for proper motion and parallax

dequote: 2016/10/25, MWB, added support for quoted semi-colon

quote: 2016/10/24, MWB, added support for quoting a semi-colon

warpstack: 2016/09/29, MWB, added MINAREA keyword

posarr: 2016/09/23, Written by Marc W. Buie, Southwest Research Institute

wrstarc: 2016/09/20, MWB, added version 1.1 for proper motion support

upsample: 2016/09/13, Written by Marc W. Buie, Southwest Research Institute

listcor: 2016/09/12, Written by Marc W. Buie, Southwest Research Institute

spexpand: 2016/09/06, Written by Marc W. Buie, Southwest Research Institute

getdigits: 2016/09/01, MWB, added support for 2-d images and improved reliability

stats: 2016/08/18, MWB, added IDX and HIST output keywords

smplprb: 2016/08/18, MWB, changed plots to black on white and fixed a minor edge effect that would cause the routine to crash.

rdsource: 2016/08/18, MWB, added support for reading srg files (same format as src).

finder: 2016/08/05, MWB, fixed a bug that caused the image to be displayed with the wrong orientation when in the horizon-lock mode. The image was actually self-consistent with the labels but was not the orientation requested.

obsprop: 2016/07/20, MWB, hardcoded change to use astronomical twilight for windows

seeing: 2016/06/24, MWB, added some extra optional plot output

starchart: 2016/06/16, MWB, changed plot algorithm to discretize the star symbol sizes and make fewer plot calls. This speeds the program up a phenomenal amount.
